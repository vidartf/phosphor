IterableOrArrayLike

IterableOrArrayLike: IIterable<T> | IArrayLike<T>
  • Defined in algorithm/iteration.ts:99

A type alias for an iterable or builtin array-like object.

Notes

The iter function can be used to produce an IIterator for objects of this type. This allows iteration algorithms to operate on these objects in a uniform fashion.

    Returns T

    The final accumulated value.

    Notes

    The reduce function follows the conventions of Array#reduce.

    If the iterator is empty, an initial value is required. That value will be used as the return value. If no initial value is provided, an error will be thrown.

    If the iterator contains a single item and no initial value is provided, the single item is used as the return value.

    Otherwise, the reducer is invoked for each element in the iterable. If an initial value is not provided, the first element will be used as the initial accumulator value.
